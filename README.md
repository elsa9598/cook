# 요리잼뱅이

요리 못하는 사람을 위한 모바일 세로형 로컬 요리 추천 앱입니다.

## 실행

```bash
npm install
npm run dev
```

## 구현된 흐름

- 스타벅스풍 따뜻한 크림 캔버스, 그린 CTA, pill 버튼, 카드 그림자 적용
- 상단 Login / 가입하기 또는 로그아웃 / 한글·영문 토글
- 모든 페이지 공통 상단 패널: Back, 작은 타이틀, 큰 타이틀
- 로컬 저장 보관함: 신선고, 얼음창고, 맛보관소, 실온바구니
- 카메라/사진, 이름, 수량, 단위, 카테고리, 메모 저장
- 양념 상세 설명, 사용 요리, 양 조절 팁
- 재료 사용 시 수량 자동 차감, 부족 재료 빨간 테두리 깜박임
- 3일 무료체험, 하루 한 번 가입 팝업, 4일째 기능 잠금
- Cloudflare Pages Function 기반 유저 생성 준비: 이메일만 저장, 비밀번호 미저장
- 가입 후 Home 메뉴가 Premium으로 전환
- 요리 추천, 장보기 목록, 선·면·도형 요리 단계 이미지, 최종 이미지 생성용 영어 프롬프트 복사

Cloudflare KV 바인딩 `YORI_USERS`를 연결하면 `functions/api/users.js`가 가입 이메일을 저장합니다. 로컬 미리보기나 KV 미연결 상태에서는 `cloudflare-ready-local` 흐름으로 표시하고, 보관함은 브라우저 로컬 저장소에만 저장합니다.
